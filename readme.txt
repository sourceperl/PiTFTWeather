Display current weather and 5 days forecast on official 7" Rpi panel.

# From

Project from repo: https://github.com/jacotronix/net.jacotronix.python.PiTFTWeather

# Use it

1. Install pywapi from https://code.google.com/p/python-weather-api/ to retrieve
data from weather.com
2. Run script with sudo python2 PiTFTWeather.py
